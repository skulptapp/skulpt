# Support

Use the route that matches the question.

| Need                                | Contact                                                                |
| ----------------------------------- | ---------------------------------------------------------------------- |
| Help using the released application | [hello@skulpt.app](mailto:hello@skulpt.app)                            |
| Problem in the released application | [errors@skulpt.app](mailto:errors@skulpt.app)                          |
| Reproducible client bug             | [GitHub Issues](https://github.com/skulptapp/skulpt/issues/new/choose) |
| Setup or architecture question      | [GitHub Discussions](https://github.com/skulptapp/skulpt/discussions)  |
| Product idea                        | [GitHub Discussions](https://github.com/skulptapp/skulpt/discussions)  |
| Vulnerability                       | [Security policy](SECURITY.md)                                         |
| Community conduct problem           | [Code of Conduct](CODE_OF_CONDUCT.md)                                  |

For an application problem, include the Skulpt version, platform version, device model, and the shortest sequence that reproduces it.

Do not post workout records, health samples, email addresses, authentication tokens, signing files, credentials, or unredacted diagnostics in a public Issue or Discussion. Send private account, deletion, SyncLayer, and data questions by email.
